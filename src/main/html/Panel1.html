This example shows you how to change the standard Hello Spiffy application and build a basic Spiffy UI navigation widget and manage panels in a Spiffy UI application.

Spiffy UI navigation panels are very flexible. They can contain headers, items, complex panels, sub-sections, and separators.

Navigation panels use CSS for layout. You can change the colors, fonts, and even the layout with simple changes to the CSS. The SpiffyUI.org CSS panel shows you an example of the navigation bar as a tab panel.
